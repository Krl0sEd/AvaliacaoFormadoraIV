# 📘 Avaliação Formadora IV — 2º Semestre

Projeto desenvolvido para a disciplina **Formadora IV**, como atividade de conclusão do **2º semestre** do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

Este repositório contém a entrega prática solicitada pela disciplina, servindo como avaliação dos conhecimentos adquiridos ao longo do semestre — incluindo lógica, organização de código, estruturação de projetos e boas práticas de desenvolvimento.

---

## 📌 Sobre o Projeto

O objetivo desta atividade é demonstrar a capacidade do aluno em:

- Aplicar conceitos estudados na disciplina  
- Desenvolver uma solução funcional e bem estruturada  
- Utilizar práticas de desenvolvimento limpas e organizadas  
- Entregar um projeto coerente com os requisitos da avaliação  

---

## 🎯 Finalidade

Este repositório funciona como **registro acadêmico** da atividade avaliativa da disciplina Formadora IV e demonstra, de forma prática, o conteúdo aprendido no 2º semestre do curso de ADS.
